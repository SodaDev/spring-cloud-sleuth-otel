2. Getting Help

If you have trouble with Spring Cloud Sleuth OTel, we would like to help.

  • Try the How-to documents. They provide solutions to the most common questions.

  • Learn the Spring Cloud Sleuth OTel basics. If you are starting out with Spring Cloud Sleuth OTel, try one of the guides.

  • Ask a question. We monitor stackoverflow.com for questions tagged with spring-cloud-sleuth-otel.

  • Report bugs with Spring Cloud Sleuth OTel at github.com/spring-cloud/spring-cloud-sleuth-otel/issues.

  • Chat with us at Spring Cloud Sleuth OTel Gitter

All of Spring Cloud Sleuth OTel is open source, including the documentation. If you find problems with the docs or if you want to improve them, please get involved.
